  [Summary]
    Unity Cubemap renderer.
--------------------------------------------------------------------------
  [Demand]
    Render a scene into a static Cubemap asset for Skybox material or
    reflective material.
--------------------------------------------------------------------------
  [Environment]
     Unity 5.0 or above.
    .Net Framework 3.5 or above.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Cubemap Renderer" in Unity editor menu bar
    and click it or press key combination Alt+C to open the editor window.

    Create/Select a scene camera as the "RenderCamera" if no main camera
    in your scene.

    Config the parameters "Face Size", "Texture Format" and "Mipmap" for
    create Cubemap.

    Click the "Render" button to open the dialog of save new render cubemap.

    Select a path and input a file name to dialog and save.
